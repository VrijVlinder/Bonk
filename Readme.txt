This plugin is based on peregrine's ExtraPage plugin, thanks to peregrine for all his work.

Bonk is an extra page to Create a custom 404 not found Page. 

After you install make sure you enable the plugin in the dashboard.

The file to edit is called bonk.php
and for extra css edits bonk.css

you can access the page http://yoursite.com/forum/plugin/Bonk

you can remove the link which appears in the menu only to admin by removing the link from the default.php

you will need to Route this page, in the dashboard go to Routes and edit what is there to: 

Default404  =  plugin/Bonk	Not Found (404)
